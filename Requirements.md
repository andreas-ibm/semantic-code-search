## Requirements
We need a system that can provide search results from our codebase that aren't a perfect match to our search term. 
### Minimum
- Semantic Search
- Index large codebases
- Multiple source codebases
- MCP API
- Containerised
- Language support:
  - C++
  - Java
  - Javascript
